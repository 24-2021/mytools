这只是一个练手的小demo，由于在实战的过程中，合成图片马很费解，一些小的doc命令参数记不住，所以就菜菜的写了一个小demo，大佬勿喷，有错误欢迎更正

输入图片的绝对路径+名称&木马的绝对路径+名称&合成图片的绝对路径+名称 
建议图片和木马放到同一个文件夹下
